# BLibs

###### Dir name ######

The Directory defines a number of transfer books

* *0001* - English
* *0002* - Spanish
* *0003* - French
* *0004* - Chinese
* *0005* - Japanese
* *0006* - Russian


###### File name ######

[ lang num ] - [ hash book ].blib

###### Syntax ######

*Title page*
```
==
! Book Name
!= Short description

!!! Year
